Logikcull Coding Challenge

For this code challenge, you are being asked to write a function called "longestSubstring" in JavaScript that takes an array of words and returns an object. Each word consists of lowercase letters ('a' − 'z') and has at least 1 character.The goal is to concatenate the words in such a way as to obtain a single word with the longest possible substring composed of one particular letter. The function returns an object with the letter and the length of the longest substring of that letter.
